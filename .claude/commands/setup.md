CLAUDE.md に、以下の内容を記載してください:
- ユーザへの返答は常に日本語で行うこと（PRの説明文なども同様）
- 適宜 docs/ のドキュメントを参照すること
- docs/ フォルダ配下は確認順序や分類に応じてファイル名の頭にプレフィックス(001, 010, 031など）をつける
- タスクの実行時は branchを作成すること
- ブランチ名は Conventional Branch に従う
- feature/[FeatureName]-[実装した機能名] 例: feature/admin-user-role-edit-invite-form
- タスク完了後にブランチの変更内容を確認し、PR作成をおこなうこと
- タスクは[ ], [x] で管理し、タスクが完了したらチェック([ ] → [x])を確実に実施すること
